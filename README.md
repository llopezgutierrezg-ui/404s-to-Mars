404s to Mars

Proyecto inspirado en la narrativa de la canción “Space Oddity”, donde un astronauta pierde el rumbo y aterriza en un punto no cartografiado de Marte. Esta experiencia recrea una pantalla de error 404 interactiva ambientada en el espacio, mezclando elementos visuales, animación y microinteracciones.

Sinopsis:
    Durante la misión, algo sale mal. El sistema de navegación falla y aparece el mensaje:

    "404 Coordenadas fuera de rango: Houston… tenemos un problema.
    El enlace que has seguido lleva a un punto no cartografiado de Marte.
    El radar no detecta señales de vida, solo polvo y eco."
    El “404” se glitchea cada ciertos segundos, indicando que el sistema continúa inestable.
    El usuario se encuentra en una interfaz espacial donde puede observar datos, paneles de control, planetas, un astronauta y un puntero con forma de puntero láser que dispara al hacer click.

Características principales:
    Panel de coordenadas (superior izquierda):
        Muestra coordenadas que cambian aleatoriamente simulando un escaneo fallido.
        Debajo del panel aparece un planeta color crema estático.

    Panel de transmisión (superior derecha):
        Indica el estado del enlace de comunicación con la nave:
            ONLINE — Enlace estable
            WEAK — Señal débil
            OFFLINE — Sin enlace
        El valor cambia dinámicamente según la "intensidad" simulada de la señal.

    Sensores de la nave (inferior izquierda):
        Pequeño panel con picos aleatorios, como un osciloscopio o radar.
        Representa los sensores intentando interpretar el entorno.

    Elementos del escenario:
        Planeta morado en la parte inferior derecha.
        Un astronauta flotando encima del planeta morado.
        Fondo animado con un video del espacio, donde las estrellas se acercan lentamente, creando sensación de movimiento y desplazamiento.

Interacción con el astronauta:
    Si el usuario acerca el cursor (que es una pistola láser) al astronauta, este mueve los brazos y la cabeza intentando esquivarlo, generando una microanimación reactiva.
    Si el usuario hace click, se dispara un rayo láser desde el puntero.

Comportamiento del mensaje 404:
    El texto “404” sufre un glitch visual cada pocos segundos.
    Refuerza la idea de fallo de navegación y señal inestable.

Botón “Volver al centro de control”:
    Representa simbólicamente la opción de reiniciar el sistema o regresar a la base.
